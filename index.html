About

PoLEJ - Polska Lista Ewaluacji Językowych

The Polish Open LLM Leaderboard aims to track, rank, and evaluate open LLMs specifically for Polish. Right now, we limit the list of models to those that can run on a single consumer GPU with up to 24GB of VRAM. We believe it helps the Polish community pick up the right models more quickly and efficiently.

If you want us to add a new model to the leaderboard or if you want to contribute, please contact us at polej@azurro.pl.

Tasks

We evaluate open language models on the following benchmarks using the Eleuther AI Language Model Evaluation Harness:
  • Belebele - The Belebele Benchmark for Massively Multilingual NLU Evaluation
  • PPC - Polish Paraphrase Corpus
  • PolEmo2 In - The PolEmo2.0 In-Domain
  • 8TAGS - The 8 tags Benchmark
  • PSC - The Polish Summaries Corpus
  • DYK - Czy wiesz?
  • LEK - Polish LEK (Lekarski Egzamin Końcowy) questions. This is a private dataset with over 5000 questions from the past few years.
For all these evaluations, we use the 0-shot approach and the greedy_until model output type. Using regex filters, we extract answers from the model's response and then calculate accuracy. A higher score indicates better performance.
